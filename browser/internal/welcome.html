Stealth Modes

The Stealth Browser respects the so-called Stealth Modes. These Modes influence the behaviour of what type of content is requested by default.

If there is additionally a site-specific mode, the site-specific mode incrementally changes the behaviour in addition to the default mode.

These site-specific modes are meant to whitelist certain domains and their dynamically generated content, for example to allow images and videos on "i.imgur.com", but not on "imgur.com".

Mode Behaviour
Offline Request only from local cache (or peer-to-peer cache)
Covert Request only texts from the internet
Stealth Request only texts and images from the internet
Online Request texts, images, videos, downloads and APIs from the internet

Keyboard Shortcuts

Binding Behaviour
[ESC] Unfocus Address Header, Hide all Sidebars
[F1] Focus Address Header, ready to type
[F2] Rotate through Stealth Modes
[F3] Rotate through Site-specific Modes and Requests
[F4] Open Stealth Browser Settings
[F5] Rotate through Tabs backwards
[F6] Rotate through Tabs forwards
[F7] Kill the first Tab
[F8] Kill the least Tab
[Ctrl]+[R] Reload current Tab
[Ctrl]+[ESC] Close current Tab
