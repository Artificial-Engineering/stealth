Site Modes

The Stealth Browser respects the so-called Site Modes. These Modes influence what type of content and media is requested from from that particular domain.

By default, the Stealth Browser will not request anything to ensure maximum control over your privacy and security.

Site Mode Request Behaviour
Requests Texts from Peers or the Internet
Requests Images from Peers or the Internet
Requests Audios from Peers or the Internet
Requests Videos from Peers or the Internet
Requests Downloads from Peers or the Internet

Keyboard Shortcuts

Binding Behaviour
[ESC] Unfocus Address Header, Hide all Sidebars
[F1] Focus Address Header, ready to type
[F2] Rotate through Stealth Modes
[F3] Toggle Site Modes sidebar
[F4] Open Stealth Browser Settings
[F5] Rotate through Tabs backwards
[F6] Rotate through Tabs forwards
[F7] Kill the first Tab
[F8] Kill the least Tab
[Ctrl]+[R] Reload current Tab
[Ctrl]+[ESC] Close current Tab
