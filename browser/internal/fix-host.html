Connect Error

The Stealth Browser could not connect to the server of the given URL. The following reasons are most likely why this happened:

  • This machine has no internet access
  • The host cache is corrupted and the server has new IP(s)
  • The server is offline

Fix Host Cache

This requires internet access. In order to make sure that the host cache is up-to-date, you can try to refresh the known host IP addresses here manually.

If they don't change, it's usually a sign that the server is offline or not reachable by its IP address.
