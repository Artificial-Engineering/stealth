Mode Error

The Stealth Browser could not allow to connect to the given URL because of the current config. The following reasons are most likely why this happened:

  • Stealth Browser is in Offline Mode
  • Site Mode does not allow to request this media type

Fix Site Mode

The Site-specific Mode will only be applied when the Stealth Browser is not in Offline Mode.

The Site Mode allows to request media types by using a whitelist-based configuration. The Site Mode incrementally overrides the defaulted Stealth Mode.
